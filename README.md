# SolidJS Templates

## Requirements
- nodejs
- pnpm

## Usage

```sh
# Download the template 
pnpx degit somerandomniko/solid-templates/TEMPLATE_NAME solidjs-template

# Change into the created directory 
cd solidjs-template

# Install dependencies
pnpm i

# Start the development server
pnpm dev
```
